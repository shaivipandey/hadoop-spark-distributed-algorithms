# hadoop-spark-distributed-algorithms
Overview
This repository contains the implementation and analysis of distributed algorithms as part of Assignment #3 for the CSE 4/587 Data Intensive Computing course at the University at Buffalo. The assignment focuses on hands-on experience with Apache Spark and Hadoop, providing practical insights into distributed data processing.

Contents
1. Hadoop Distributed File System (HDFS) Review: A comprehensive review of the "Hadoop Distributed File System" paper by Konstantin Shvachko, Hairong Kuang, Sanjay Radia, and Robert Chansler.
2. Dijkstra's Shortest Path Algorithm: Implementation of the Dijkstra's algorithm using Apache Spark, applied to a graph represented in the provided dataset.
3. PageRank Algorithm: Implementation of the PageRank algorithm to analyze a simulated network of 100 pages with hyperlinks, determining the rank of each page.
